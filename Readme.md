# Introduction
This backend for Coding platform where participants can solve questions for the problems provided, run the questions using the Sphere Engine and the admin can add, edit or delete the questions.

## Tech Stack
<ol>
    <li>ExpressJS</li>
    <li>BcryptJS</li>
    <li>MongoDB</li>
    <li>Winston</li>
    <li>Docker</li>
    <li>nodemailer</li>
    <li>jsonwebtoken</li>
</ol>
